# Changelog

## 1.1.0 - 2026-06-26

### Added

- 新增协作者只读同步页：可扫描本地同步目录，对比最新资源表，统计有效、缺失、过期和额外本地文件。
- 新增独立同步目录设置，和普通下载目录分开保存。
- 新增同步范围选择：支持同步全部资料，或只同步左侧当前选中的目录。
- 新增打开同步目录入口，复用后端安全目录打开逻辑。
- 新增大列表虚拟滚动：资料列表、下载队列、同步待处理列表和额外本地文件列表在大量文件时保持可操作。
- 新增下载队列筛选和同步待处理筛选，方便只查看进行中、失败、完成、缺失或过期项目。

### Changed

- 下载进度行更新改为按 animation frame 合并，降低大量文件或高频进度事件时的界面卡顿。
- 下载页和同步页的密集列表布局调整为稳定可滚动区域，避免长列表挤压其他控件。
- 左侧索引栏现在将图标、标题和 `全部资料` 固定在顶部，仅目录索引区域独立滚动。

### Fixed

- 修复暗色模式下页面超过窗口高度的底层背景仍保持亮色的问题。
- 修复左侧索引内容过长时可能撑开整体布局并影响右侧部件展示的问题。

### Notes

- 本版本仍只发布 Windows 和 Linux 构建，不包含 macOS 构建。
- 安装包仍未进行代码签名，Windows 可能出现 SmartScreen 提示。
- 协作者仍需要本地配置 rclone 只读 R2 remote，并克隆 `TYUT-ebooks-collection-neo` 索引仓库。
- 同步功能只会下载缺失或过期文件，不会上传、删除或移动本地额外文件。
